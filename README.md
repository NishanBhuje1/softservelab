# SoftServe Lab

Company landing page for SoftServe Lab, my registered Australian business where I build and ship indie mobile and web applications.

**Live site:** [softservelab.com](https://softservelab.com)

## About

SoftServe Lab is the studio name under which I develop and launch production applications. Current shipped projects include TwoWallet (Flutter couples-finance app), Goldmark (full-stack e-commerce platform), and FixMate Mobile (live client website for a Sydney phone repair business).

The next update to this site will include a portfolio showcase of these shipped projects.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Deployed on Vercel

Simple, fast static site. Clean company presence with room to grow as the portfolio expands.
